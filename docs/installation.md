# Installation

using `uv` (recommended):

```bash
uv add git+https://github.com/MCRE-BE/ntfy_lite.git
```

from source:

```bash
git clone https://github.com/MCRE-BE/ntfy_lite.git
cd ntfy_lite
uv pip install .
```

or

```bash
uv pip install git+https://github.com/MCRE-BE/ntfy_lite.git
```
